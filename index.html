PCIT Project

Panther Compiler Infrastructure and Toolchain, and the home of the Panther programming language. PCIT (pronounced "P-Kit") relies on the LLVM-Project, but that is not guaranteed forever. It is written entirely in C++, but a self-hosted version (written entirely in Panther) is planned.

Tools

All tools can be used as stand-alone programs or as libraries.

Panther / pthr

Statically typed, compiled, high-performance, general-purpose programming language. See more here.

Other compilers

Will also be able to compile C and hopefully C++ code. The main intent is for this to allow seamless interoperability with Panther, but these frontends will be able to be used stand-alone.
