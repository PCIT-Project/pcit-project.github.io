Operators

Panther
Documentation


  Operators are a set of operations that create an expression from one or more sub-expressions. Unless remarked otherwise, all operators support operator overloading and forwarding of fluid values. In addition, all operators are comptime (unless they are overloaded, in which case it is overload specific).

Pointer/Optional Operators

Name Syntax Supported Builtin Types Example Remarks
Address Of
 &a

Any

 var num: Int = 12;
const num_ptr: Int* = &num;
num_ptr.* // 12

Argument must be concrete.

Using on a mutable object returns a mut-qualified pointer,
otherwise is a pointer.
Dereference
 a.*

Any pointer

 var num: Int = 12;
const num_ptr: Int* = &num;
num_ptr.* // 12

Dereferencing a mut-qualified pointer results in a
mutable value, otherwise the value is constant.
Unwrap
 a.?

Any optional

 var opt_num: Int? = 12;
opt_num.? // 12

It is undefined behavior to unwrap an optional
that is null.
