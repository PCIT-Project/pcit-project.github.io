Atomic Intrinsics

Panther
Documentation


@atomicLoad

func @atomicLoad = <{TARGET: Type, OUTPUT: Type, ORDER: @pthr.AtomicOrdering}> (target: TARGET) -> OUTPUT;

Atomic load. TARGET must be a non-optional pointer. TARGET must be the result of a dereference of TARGET.

@atomicStore

func @atomicStore = <{TARGET: Type, VALUE: Type, ORDER: @pthr.AtomicOrdering}> (target: TARGET, value: VALUE in) -> Void;

Atomic load. TARGET must be a non-optional pointer. TARGET must be the result of a dereference of TARGET.

@cmpxchg

func @cmpxchg = <{TARGET: Type, VALUE: Type, IS_WEAK: Bool, SUCCESS_ORDER: @pthr.AtomicOrdering, FAILURE_ORDER: @pthr.AtomicOrdering}> (target: TARGET, expected: VALUE, desired: VALUE in) -> (loaded: VALUE, succeeded: Bool);

Atomic compare-exchange. TARGET must be a non-optional pointer. TARGET must be the result of a dereference of TARGET. Return parameter loaded is the value of target that was atomically loaded. Return parameter succeeded is true if the exchange was successful.
