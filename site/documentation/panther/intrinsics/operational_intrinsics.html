Operational Intrinsics

Panther
Documentation


@abort

func @abort = () -> Void;

Abort the program. Lowers to a trap instruction if the target has one, otherwise lowered to LIBC abort().

@breakpoint

func @breakpoint = () -> Void;

Cause an execution trap to request the debugger break execution.

@makeInitPtr

func @makeInitPtr = (uninit_ptr: $T*!) -> T*;

Converts an uninitialized-qualified pointer to a pointer. Paramter uninit_ptr must be concrete, a local variable, and the pointee must be known to be initialized.
